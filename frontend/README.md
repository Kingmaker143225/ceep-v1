# 🚀 TGECET 2026 - Run Procedure

## 1️⃣ Clone Repository

```bash
git clone https://github.com/gitcognisysai/tgecet_2026.git
```

---

## 2️⃣ Move Into Project Folder

```bash
cd tgecet_2026
```

---

## 3️⃣ Switch to Dev Branch

```bash
git checkout dev
```

If dev branch does not exist locally:

```bash
git checkout -b dev
```

---

## 4️⃣ Install Dependencies

```bash
npm install
```

---

## 5️⃣ Run Development Server

```bash
npm run dev
```

---

## 6️⃣ Open in Browser

After running the above command, terminal will show:

```bash
Local: http://localhost:5173/
```

Open that URL in browser.

---

# 🛠️ Build for Production

```bash
npm run build
```

---

# 👀 Preview Production Build

```bash
npm run preview
```

---

# 🔄 Push Latest Code to Dev Branch

```bash
git add .
git commit -m "updated project"
git push origin dev
```
